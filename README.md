# 📚 Book Manager

A modern Book Management application built with **React + TypeScript + Tailwind CSS**.  
Users can browse, search, and add new books with a clean and responsive UI.

---

## ✨ Features

- 🔍 Search books by title or author
- ➕ Add new books with validation
- 📖 Book detail navigation
- 🎨 Smooth hover animations
- ⚡ Simulated loading state
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React
- TypeScript
- React Router
- Tailwind CSS
